# 数据要素大赛 — 项目结构

```
数据要素大赛/
├── app.py                          # Streamlit Web 界面（5个Tab）
│   ├── Tab1: 单篇验证
│   ├── Tab2: 批量验证（CSV + PDF导出）
│   ├── Tab3: 中文文献验证（知网/万方引用自动解析）
│   ├── Tab4: 文件导入（PDF/BibTeX）
│   └── Tab5: 自检测试（真假得分对比）
│
├── verify_engine.py                # 核心验证引擎（1054行）
│   ├── 数据模型: ReferenceRecord / VerificationResult
│   ├── 级联验证: CrossRef → OpenAlex → Semantic Scholar
│   ├── 8个验证维度: DOI格式 + 语义一致性 + 数据源 + 作者匹配
│   │                + AI特征 + 期刊匹配 + DOI真实性 + 元数据一致性
│   ├── 6档分类: 确定真实 / 高度可信 / 存疑-可能为真
│   │           / 存疑-可能为假 / 高度存疑 / 确定虚假
│   ├── RapidFuzz 模糊匹配（token_sort_ratio / partial_ratio）
│   ├── 12项 AI 标题特征检测（100% accuracy / NP-complete 等）
│   ├── SQLite 持久化缓存 + 离线模式
│   ├── 中英文语义一致性检查（SemanticChecker）
│   ├── 知网/万方引用解析（parse_references_batch）
│   ├── PDF 参考文献提取（PyMuPDF）
│   ├── BibTeX 解析（bibtexparser）
│   ├── PDF 验证报告生成（reportlab）
│   ├── 元数据修正建议（_suggest_corrections）
│   ├── 3线程并发批量验证（verify_batch_concurrent）
│   └── 自测试套件（run_self_test）
│
├── demo.ipynb                      # Jupyter Notebook 演示
├── streamlit_app.py                # Streamlit Cloud 部署入口
├── requirements.txt                # Python 依赖清单（10个包）
├── README.md                       # 项目使用说明
├── LICENSE                         # MIT 开源协议
├── .gitignore
│
├── .streamlit/
│   └── config.toml                 # 暗色主题 + 上传限制100MB
│
├── .github/workflows/
│   └── ci.yml                      # GitHub Actions CI（18个pytest）
│
├── data/
│   ├── demo_data.csv               # 演示数据（6真 + 3假文献）
│   └── template.csv                # 批量验证CSV模板
│
├── tests/
│   ├── __init__.py
│   └── test_verify_engine.py       # 18个单元测试
│       ├── TestSemanticChecker (5)
│       ├── TestDOICheck (3)
│       ├── TestTitleAIFingerprints (2)
│       ├── TestCascadeVerify (3)

│       ├── TestBatchConcurrent (1)

└── docs/
    ├── specs/
    │   ├── 2026-06-21-lit-verify-design.md        # 原始设计文档
    │   └── 2026-06-25-v3-fine-grained-cascade-design.md  # v3设计文档
```

## 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | Streamlit + Plotly |
| 验证引擎 | Python 3.12 + RapidFuzz |
| API 数据源 | CrossRef / OpenAlex / Semantic Scholar |
| PDF 解析 | PyMuPDF (fitz) |
| BibTeX 解析 | bibtexparser |
| PDF 报告 | reportlab |
| 缓存 | 内存 dict + SQLite |
| 测试 | pytest (18 cases) |
| CI/CD | GitHub Actions |
| 部署 | Streamlit Cloud |

## 约束

- 不涉及模型训练 / 深度学习
- 仅调用公开免费 API（无需 Key）
- 批量验证 3 线程并发
- 评分 0-100，6 档细粒度分类

## 运行方式

```bash
# 本地运行
pip install -r requirements.txt
streamlit run app.py

# 运行测试
pytest tests/ -v

# Jupyter 演示
jupyter notebook demo.ipynb
```
